## 데이터가 존재하는지 확인할 때 사용하는 컬렉션

- `List` 의 `contains` 는 `O(N)` 만큼의 시간이 걸립니다.
- `Set` 의 `contains` 는 `O(1)` 만큼의 시간이 걸립니다.

데이터를 단순 저장하고 존재하는 지 확인할 때 `index`로 하지 않을 경우에는 `Set`을 사용하는게 시간을 단축할 수 있습니다.
