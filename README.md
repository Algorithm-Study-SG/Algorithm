커밋 규칙 

1. Repository Clone

```
git clone https://github.com/Algorithm-Study-SG/algorithm.git
```

2. 본인의 branch 생성

```
git checkout -b {자신의 깃허브 이름}/{주차}
```

3. git
```
git add .
git commit -m "{주차} : {문제번호}"
git push
```

4. 스터디 회의 후, main branch에 merge
